1. git config --global user.email "chania.eva@gmail.com"
digunakan untuk memperkenalkan diri email kepada git.
2. git config --global user.name "Chania"
digunakan untuk memperkenalkan nama kepada git.
3. git config --list
digunakan untuk melihat konfigurasi yang kita masukkan ke git.
4. git init 
digunakan untuk mengisiasikan suatu folder supaya terhubung dengan git.
5. git add
digunakan untuk menginisiasikan suatu file ke dalam git
6. git status
digunakan untuk memeriksa apakah suatu file sudah berhasil masuk ke dalam git
7. git commit -m (argumen) "(Nama)"
digunakan untuk save file ke git
8. git log
digunakan untuk melihat commit yang sudah pernah tersimpan
9. git reset (HASH)
digunakan untuk kembali ke commitan sebelumnya (harus pake discard)
10. git reset --hard
digunakan untuk discard (tanpa discard)

Warna merah : baris dihapus
Warna Biru : baris diubah
Warna hijau : belum disimpan

Cara add dan commit secara cepat (Visual Studio Code):
1. Buka ekstensi Source Control
2. untuk git add, klik tombol plus disebelah file yang berubah
3. commit dengan menjelaskan apa saja yang dilakukan secara sederhana
